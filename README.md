# Pechkin

Монорепозиторий с frontend и backend сервисами.

## Структура

```
pechkin/
├── packages/
│   ├── frontend/    # Фронтенд приложение
│   └── backend/     # Бэкенд API
```

## Быстрый старт

```bash
# Установка зависимостей
npm install

# Запуск всех сервисов в dev режиме
npm run dev

# Сборка всех сервисов
npm run build
```

## Требования

- Node.js 18+
- npm 9+
